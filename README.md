A fully immersive, cyberpunk-themed developer portfolio. Zero frameworks. Zero templates. Every pixel hand-crafted with vanilla HTML, CSS, and JavaScript to deliver a "Hacker Terminal" experience — from CRT scanlines and glitch effects to holographic ID badges and encrypted text.

---

## ⚡ Tech Stack

| Layer | Technology |
|---|---|
| **Structure** | HTML5 (Semantic) |
| **Styling** | CSS3 — Custom Properties, Flexbox, Grid, 3D Transforms, `perspective`, `clip-path` |
| **Logic** | Vanilla JavaScript (ES6+) |
| **Animations** | CSS Keyframes (Glitch, CRT Flicker, Scanner Beams, Port Flicker) |
| **Scroll** | Intersection Observer API — staggered reveal animations |
| **Canvas** | HTML5 Canvas — animated radar chart + floating particle network |
| **Forms** | Fetch API (AJAX) → Formspree endpoint — zero page redirects |
| **Fonts** | Google Fonts — Orbitron, Rajdhani, Share Tech Mono |
| **Dependencies** | **None.** 100% bespoke code. |

## 🔥 Key Features

### `[SYS]` Operator Dossier (About)
- Cyber profile grid with `SUBJECT // 0xDILEEP` identity header
- Hex-scrambled bio text (`0xA4 0x8F...`) that **decrypts on hover** with CRT flicker

### `[OP]` Intel Archive (Projects)
- Alternating HUD datapad layout with **full project images**
- Images load with grayscale + sepia + scanline overlays → snap to full color on hover
- Mechanical targeting corners animate on interaction
- HUD classification badges: `WEB_APP // SECURE`, `AI // MACHINE_LEARNING`

### `[NET]` Corporate Contracts (Experience)
- Bounty Board grid — asymmetrical digital dossiers
- Cryptic code names: `OP: TAP_ACADEMY`, `OP: WEB_DEV_INIT`
- Hover-triggered glitch effect reveals **Mission Debrief** bullet points

### `[MEM]` Biometric Clearance (Education)
- 3D-perspected holographic ID card (`rotateY`, `rotateX`, `translateZ`)
- Animated **laser scanner beam** sweeps on hover
- Wireframe grid background fades in during scan
- Fields: `CLEARANCE LEVEL: BACHELORS` / `SECTOR: INFORMATION SCIENCE`

### `[COM]` Secure Transmission (Contact)
- Bash-style terminal form: `guest@system:~$ set EMAIL=`
- AJAX submission via Formspree — **no page redirect**
- Real-time feedback: `> TRANSMISSION SUCCESSFUL.` or `> ERR: FATAL NETWORK EXCEPTION.`

### Global Theming
- 🎯 Custom `::-webkit-scrollbar` — cyan thumb, magenta on hover
- 🎯 Custom `::selection` — magenta background, yellow text
- 🎯 Persistent CRT scanline overlay across entire DOM
- 🎯 Floating particle network with dynamic connection lines
- 🎯 Navbar filesystem navigation: `./about` `./projects` `./contracts` `./contact`
- 🎯 Typing effect cycles: `Java Full Stack Developer` → `NEURAL_NET.ARCHITECT` → `ZERO_BUG_PROTOCOL // ACTIVE` → `SYS.WEB.ENGINEER`
